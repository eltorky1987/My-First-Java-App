# ☕ My First Java App | أول تطبيق جافا  

## 📖 الوصف | Description  
مشروع تجريبي بسيط لتعلم أساسيات لغة **Java**.  
يهدف إلى تجربة كتابة أول برنامج وتشغيله باستخدام بيئة تطوير مثل Eclipse أو IntelliJ.  

---

## 🛠️ المكونات | Components  
- لغة Java (JDK)  
- بيئة تطوير (Eclipse / IntelliJ IDEA)  
- ملفات المشروع الأساسية  

---

## 💻 الملفات | Files  
- `MainActivity.java` → الملف الرئيسي للتطبيق.  
- ملفات إضافية للتجارب أو التوسعات.  

---

## ⚙️ طريقة التشغيل | How to Run  
1. افتح المشروع في بيئة التطوير (Eclipse أو IntelliJ).  
2. تأكد من تثبيت **JDK** على جهازك.  
3. شغل الملف `MainActivity.java` لرؤية النتيجة.  

---

## 📡 التوسع المستقبلي | Future Expansion  
- إضافة واجهة رسومية (GUI) باستخدام Swing أو JavaFX.  
- تجربة ربط التطبيق بقاعدة بيانات صغيرة.  
- تطوير المشروع ليصبح تطبيق عملي أكبر.
